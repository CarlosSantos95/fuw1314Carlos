iTalc Master

Instalamos iTalc Master en un equipo GNU/Linux, en este caso Debian 7. En GNU/Linux la instalación de italc-master crea los siguientes ficheros: /etc/italc/keys/private/teacher/key /etc/italc/keys/public/teacher/key.
En este caso,al instalar el programa,no teniamos las claves públicas ni privadas creadas,por lo que realizamos este comando para generarlas.
-createkeypair



Creamos el grupo italc,creamos el usuario "maul" y lo añadimos a nuestro grupo:



Damos permisos al grupo italc, para tener acceso a las claves de italc: cd /etc/italc; chgrp -R italc keys



Verificar que el servicio está iniciado (ps -ef |grep ica). Si el programa "ica" no está iniciado, lo podemos iniciar manualmente. Para ello abrimos un terminal y escribimos: /usr/bin/ica &




Para que el servicio ica se inicie automáticamente al iniciar sesión con el usuario "maul":
Editar fichero /home/maul/.profile Añadir "/usr/bin/ica &" al final del fichero.



iTalc Cliente Windows

Primero tenemos que asegurarnos de que nuestro usuario de Windows tiene una clave NO vacía. A continuación tenemos que instalar una versión del iTalc en Windows similar a la versión Master.


Una vez comprobado lo anterior,procedemos a instalarlo.La instalación se realiza sin ningún problema.
Creamos las claves, buscamos el fichero "key" dentro de C:\Archivos de programas\italc\key\public\teacher.Lo sustituimos por el que nos da el master.Por último reiniciamos el programa

iTalc Cliente Linux

Instalamos en GNU/Linux Debian (apt-get install italc-client) Copiamos el fichero key del master directamente en /etc/italc/keys/public/teacher. Creamos el grupo italc, y añadimos nuestro usuario al grupo:

addgroup italc usermod usuario --groups italc

Damos permisos al grupo italc, para tener acceso a las claves de italc:
cd /etc/italc; chgrp -R italc keys

Verificamos que el servicio está iniciado (ps -ef |grep ica). Si el programa ica no está iniciado, iniciarlo manualmente.
Abrimos terminal y escribimos: /usr/bin/ica & Para que el servicio ica se inicie automáticamente al iniciar sesión con el usuario "maul"
Editamos fichero /home/maul/.profile
Añadimos "/usr/bin/ica &" al final del fichero.
